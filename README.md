# words_memorizer
Simple script to memorize foregin language words and phrases.

## Run

`./memwords.py <file with words>`

Example:  
`./memwords.py  espanol_numerales.mem`

## How to do exercises

You need to inpute translation of asked word aor phrase.

```
ask    |  tres
answer |  три
       |  ok - 1
-------+------------------------------
ask    |  восемь
answer |
```
